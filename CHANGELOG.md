#### Breaking Changes

* error response structure is changed

    To migrate the code follow the example below:

    Before:

    ```js
    {
        error: true,
        message: ‘’,
        resolvedMessages: []
    }
    ```

    After:

    ```js
    {
        message: ‘’,
        errors: {
            ‘fieldName’: ‘error message’,
            ‘fieldName2’: [‘error message’, ‘error message 2 ’] // if errors are more than 1
        }
    }
    ```
